# Flight Records Analysis (Power BI)

## Project Overview
- An interactive Power BI Dashboard analyzing 1.9+ million flight records
- Focused on performance, delays, and aircraft efficiency

## Data Preparation
- Cleaned and transformed data using Python (Pandas) in Jupyter Notebook
- Modeled data into Fact and Dimension tables
- Applied a Star Schema for analytics and reporting

## Dashboard Highlights
- Total airtime by month
- Average arrival & departure delays
- Average aircraft speed
- Slicers by:
  - Aircraft
  - Date
- Identified:
  - Fastest aircrafts
  - Slowest aircrafts

## Tools & Technologies
- Power BI
- Python
- Pandas
- Jupyter Notebook
- OS (file handling)
